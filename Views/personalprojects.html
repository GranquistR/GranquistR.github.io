Tiny Blocks

Tiny Blocks is a game I made in Visual Studio using the C# windows form template. The game consists of a three-dimensional isometric grid of blocks where the player can create and destroy to their heart's desire. The game uses low density bitmaps and occlusion culling to limit the amount of drawing done on the screen which allows for large max grid sizes while keeping a smooth frame rate. You control a cursor that can move in all 6 directions in 3D space, use the cursor to break or place the blocks of all varieties at your cursors location.
