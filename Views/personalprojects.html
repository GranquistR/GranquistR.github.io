Tiny Blocks

Tiny Blocks is a game I made in Visual Studio using the C# windows form template. The game consists of a three-dimensional isometric grid of blocks where the player can create and destroy to their heart's desire. The game uses low density bitmaps and occlusion culling to limit the amount of drawing done on the screen which allows for large max grid sizes while keeping a smooth frame rate. You control a cursor that can move in all 6 directions in 3D space, use the cursor to break or place the blocks of all varieties at your cursors location.

This Website

My intent with this website, besides being an awesome portfolio, was to have it serve as a hub to all future personal projects. This would entail making the page as fast and expandable as possible. I achieved this by making my own single page application design framework to work off of. This means that the page is loaded once and then all future page loads are done by javascript. This allows for a seamless transition between pages and allows for the page to be loaded once and then never again. Aiding this design is a vue-like component system that involves defining components in their own .HTML file and loading them with javascript.
